# delete-cell-english

---
## 功能：
1.   删除输入文件中指定sheet页内的中英文混合单元格的英文。
2.   读取config中的中英文对照关系，若当前excel存在该对应关系则添
加翻译

---

## 使用方法

1. 按照自己的需求编辑config文件
2. 运行 MainPage.py 
3. 执行删除功能后，工作目录下的*_copy.xlsx为目标文件
4. 执行翻译功能后，工作目录下的*_translate.xlsx为目标文件
